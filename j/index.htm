BACKGROUND-BLEND-MODE

This property allows you to blend multiple background images or colors together, creating interesting visual effects. You can use different blend modes like multiply, screen, overlay, and more.

.abc {
  background-image: linear-gradient(to right, red, blue);
  background-color: yellow;
  background-blend-mode: multiply;
}

TEXT-ALIGN

This property aligns the text within an element. You can use values like left, right, center, and justify to control the alignment.

.abc {
  text-align: center;
}
