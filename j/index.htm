BACKGROUND-BLEND-MODE

This property allows you to blend multiple background images or colors together, creating interesting visual effects. You can use different blend modes like multiply, screen, overlay, and more.

.xy {
  background-image: linear-gradient(to right, red, blue);
  background-color: yellow;
  background-blend-mode: multiply;
}

BORDER-RADIUS

This property allows you to round the corners of an element's border. You can specify different radii for each corner, or a single value to apply to all corners.

.xy {
  border-radius: 10px;
}

BOX-SHADOW

This property adds a shadow effect to an element, making it appear to be raised or lowered. You can control the color, offset, blur, and spread of the shadow.

.xy {
  box-shadow: 0px 5px 10px rgba(0, 0, 0, 0.2);
}

CLIP-PATH

This property allows you to create complex shapes for elements by clipping them to a specific region. You can use various shape functions and paths to define the clipping area.

>.xy {
  clip-path: circle(50% at 50% 50%);
}

OPACITY

This property sets the transparency of an element. A value of 1 is fully opaque, while 0 is fully transparent.

.xy {
  opacity: 0.5;
}

TEXT-ALIGN

This property aligns the text within an element. You can use values like left, right, center, and justify to control the alignment.

.xy {
  text-align: center;
}

TRANSFORM

This property allows you to manipulate the position, size, rotation, and perspective of an element. You can use various transformation functions like translate, scale, rotate, and skew.

.xy {
  transform: rotate(45deg);
}
